To install SharePoint Framework (SPFx) software:-
Open Windows Powershell:-
install-module MSOnline
install-module Microsoft.Online.Sharepoint.Powershell
install-module SharepointPnPPowershellOnline

For command help:-
get-help *mso* (For MS Online command)
get-help *spo* (For Sharepoint command)
get-help *pnp* (For PnP command)

Install Node.js and Visual Studio Code

Open Node.js command prompt for supporting software:-

npm install -g yo gulp
npm install -g @Microsoft/generator-sharepoint

yo @microsoft/sharepoint (To Create Sharepoint Project)
gulp -trust-dev-cert (To create certificate in case of certificate error)
gulp serve (To run Webpart application)
